## Usage

```bash
next log
```

## Description

Displays the contents of LOG.md — an append-only record of completed tasks.

## Output

```markdown
# Task Log

Completion history for my-project

## 2026-02-03

- **T1**: Initialize project ✅ `abc1234` @ 14:30
- **T2**: Add core types ✅ `def5678` @ 14:45
- **T3**: Implement service ✅ `ghi9012` @ 15:02
```
