# next init

Initialize a project for next-task.

## Usage

```bash
next init
```

## Creates

| File | Purpose |
|------|---------|
| `.agent/config.yml` | Configuration |
| `.agent/state.json` | State tracking |
| `LOG.md` | Completion history |

## After Init

Create a `TASKS.md` file with your tasks, then run `next status`.
