# sprint status

Show task queue status.

## Usage

```bash
sprint status
sprint s
```

## Output

```
📋 my-project — Task Status

Done:
  ✅ T1: Initialize project (abc1234)
  ✅ T2: Add core types (def5678)

In Progress:
  🔄 T3: Implement service

Ready:
  T4: Add API routes @codex

Blocked:
  T5: Integration tests (waiting: T4)

Done: 2 | In Progress: 1 | Ready: 1 | Blocked: 1
```
